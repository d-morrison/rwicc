# Seroconversion density function

Probability density, `survival * hazard`.

## Usage

``` r
seroconversion_density_function(...)
```

## Arguments

- ...:

  arguments passed to
  [`seroconversion_survival_function()`](https://d-morrison.github.io/rwicc/reference/seroconversion_survival_function.md)
  and
  [`seroconversion_hazard_function()`](https://d-morrison.github.io/rwicc/reference/seroconversion_hazard_function.md)

## Value

a numeric vector of density values
